# Fastline Logistics 🚛

Fastline Logistics is a consignment tracking website that allows users to track their shipments in real-time.

## Features
- Track shipments using tracking number
- View sender and receiver details
- Shipment timeline and status updates

## How to Use
1. Enter your tracking number
2. Click "Track Now"
3. View shipment details instantly
